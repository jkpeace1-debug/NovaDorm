# 노바돔 (Novadorm) - 청년 글로벌 기숙사

## 프로젝트 개요

노바돔은 외국인 유학생과 국내 청년을 위한 안전하고 편리한 글로벌 기숙사 플랫폼입니다. 홍대 중심부에 위치한 1호점을 시작으로, 청년들의 주거 문제를 해결하고 글로벌 커뮤니티를 형성합니다.

## 🏠 주요 특징

### 1. **스타벅스 인테리어 컨셉**
- 따뜻한 우드톤과 적절한 그린 포인트
- 과도한 녹색 피하고 내추럴한 색상 사용
- 고급스럽고 편안한 분위기

### 2. **서비스 중심의 단순한 디자인**
- 복잡한 투자 정보 제거
- 부모님이 궁금해하실 정보만 담음
- 깔끔하고 모던한 인터페이스

### 3. **K-Care 구독 시스템**
- NovaDorm 기본 서비스는 그대로 유지
- 추가 케어가 필요한 경우 선택 가능
- Basic/Standard/Premium 3단계 구독 모델

### 4. **핵심 섹션 구성**
- **홈**: 서비스 소개 및 핵심 메시지
- **문제/솔루션**: 청년들의 주거 문제와 해결책
- **케어/안전**: 24시간 보안 시스템, 상주 관리인, K-Care 구독
- **시설/서비스**: 편리한 생활을 위한 시설
- **프로그램**: 언어 교환, 문화 체험 등
- **부모 정보**: 학부모님들을 위한 FAQ 및 상담
- **위치/문의**: 홍대 중심부의 편리한 위치
- **Coming Soon**: 향후 확장 계획

### 5. **NovaAgency 연동**
- 상단 메뉴에 NovaAgency 링크 추가
- www.novaskr.com으로 연결

### 6. **부모 중심 접근**
- 자녀의 안전과 생활 정보 제공
- 24시간 관리인 상주
- 정기적인 생활 상담
- 의료 지원 시스템

### 7. **투자 정보 최소화**
- 상단 우측 "Investor Relations" 링크만 제공
- 별도의 IR 페이지에서 간단한 정보만 제공
- IR 문의 기능만 구현

## 🎯 타겟 고객

- **외국인 유학생**: 안전한 주거 공간이 필요한 유학생
- **국내 청년**: 합리적인 가격의 기숙사를 찾는 청년
- **학부모님**: 자녀의 안전한 생활을 걱정하시는 부모님

## 📍 위치 정보

- **1호점**: 서울특별시 마포구 홍대입구역 도보 5분
- **교통**: 지하철, 버스 등 대중교통 이용 편리
- **인근 시설**: 이마트, 병원, 공원 등 생활 인프라 완비
