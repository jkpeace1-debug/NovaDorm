# NovaDorm 홈페이지 관리 가이드

## 완성된 기능

### 1. 메인 홈페이지
- **스타벅스 인테리어 컨셉**: 따뜻한 우드톤과 적절한 그린 포인트
- **문제/솔루션 섹션**: 청년들의 주거 문제와 해결책
- **케어/안전 섹션**: 24시간 보안, 상주 관리인, 생활 케어, 건강 관리
- **K-Care 구독 시스템**: Basic/Standard/Premium 3단계 구독 모델
- **시설/서비스**: 개인 침실, 공용 주방, 세탁실, 인터넷, 공용 라운지, 운동시설
- **프로그램**: 언어 교환, 문화 체험, 학업 지원, 네트워킹
- **부모 정보**: FAQ, 상담 서비스
- **위치/문의**: 홍대 위치 정보, 입주 문의 폼
- **Coming Soon**: 성수 2호점, 모바일 앱, 온라인 프로그램

### 2. 투자자 관계 페이지
- 간단한 회사 소개
- IR 자료 요청 폼

### 3. 외부 연동
- NovaAgency (www.novaskr.com) 링크

## 홈페이지 관리 방법

### 1. 기본 수정 방법

#### 텍스트 수정
1. `index.html` 파일을 열기
2. 찾고자하는 텍스트 검색 (Ctrl+F)
3. 직접 텍스트 수정
4. 저장

#### 이미지 변경
1. 이미지 URL을 찾아서 새 URL로 교체
2. 또는 이미지 파일을 다운로드하여 프로젝트에 추가

#### 색상 변경
1. `css/style.css` 파일 열기
2. `:root` 섹션의 색상 변수 수정
3. 저장

### 2. 파일 구조
```
index.html              # 메인 페이지
investor.html           # 투자자 관계 페이지
css/
  └── style.css       # 스타일시트
js/
  └── main.js         # 자바스크립트
website-guide.md        # 이 관리 가이드
```

### 3. 주요 섹션 위치

#### 네비게이션 메뉴
- 위치: `index.html` 20-35번째 라인
- 수정: `<ul class="nav-menu">` 안의 `<li>` 항목들

#### 히어로 섹션 (메인 비주얼)
- 위치: `index.html` 40-72번째 라인
- 제목: `<h1 class="hero-title">`
- 설명: `<p class="hero-description">`

#### K-Care 구독 섹션
- 위치: `index.html` 150-200번째 라인 (변경될 수 있음)
- 구독 플랜: `.subscription-plans` 클래스 안

#### 푸터
- 위치: `index.html` 420-450번째 라인
- 연락처, 운영시간 등

## 배포 방법

### 1. 발행 탭 사용
1. 웹사이트 우측 상단의 "Publish" 탭 클릭
2. "Publish Website" 버튼 클릭
3. 자동으로 배포 완료

### 2. 도메인 설정
- 발행 후 제공되는 URL 사용
- 또는 커스텀 도메인 연결 가능

### 3. 결제 처리
- 현재는 문의만 받는 형태
- 실제 결제 시스템 연동은 별도 개발 필요

## K-Care 구독 관리

### 구독 플랜
1. **Basic**: 월 29,000원
   - 월 1회 건강 체크
   - 기본 상담 서비스
   - 세탁 서비스 2회

2. **Standard**: 월 49,000원 (인기)
   - 주 1회 건강 체크
   - 전문 상담 서비스
   - 세탁 서비스 4회
   - 의료 지원 및 동행

3. **Premium**: 월 79,000원
   - 주 2회 건강 체크
   - 무제한 세탁 서비스
   - 24시간 응급 의료
   - 개인 맞춤 영양 관리

## 주의사항

1. **신용카드 크레딧**: 단순한 텍스트 수정, 이미지 변경, 색상 조정은 크레딧이 소모되지 않습니다.

2. **백업**: 중요한 수정 전 현재 상태를 저장해 두는 것을 권장합니다.

3. **테스트**: 수정 후 항상 웹사이트가 정상적으로 작동하는지 확인합니다.
